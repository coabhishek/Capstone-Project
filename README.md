# Little Lemon Restaurant - Capstone Project

A dynamic table booking web application built as the Capstone Project for the Meta Front-End Developer Professional Certificate. The application allows users to reserve a table at the Little Lemon restaurant with real-time slot availability.

## 🚀 Live Demo
Check out the live application here: [Live Deployment Link]([YOUR_DEPLOYMENT_LINK])

## ✨ Key Features
- **Dynamic Routing:** Implemented seamless navigation between Homepage, Booking Page, and Confirmation Page using `react-router-dom`.
- **State Management with useReducer:** Efficiently managed complex state for dynamic time-slot availability based on user-selected dates.
- **Mock API Integration:** Embedded safe fallback functions (`fetchAPI` and `submitAPI`) to mimic asynchronous data fetching and form submissions.
- **Form Validation & Submission:** Handles table booking data seamlessly and safely redirects users to a custom confirmation screen upon successful booking.
- **Responsive Layout:** Completely mobile-friendly UI crafted with CSS for smooth cross-device user experience.

## 🛠️ Tech Stack Used
- **Frontend:** React.js (Functional Components, Hooks)
- **State Management:** React `useReducer` Hook
- **Routing:** React Router v6
- **Styling:** Custom CSS (including Flexbox, Grid, Animations, and Media Queries)
